Welcome to DeepPathway's documentation!
===========

DeepPathway is a deep learning framework for predicting pathway expression from H&E images. It uses spatial transcriptomics data to train a model using bimodal contrastive learning approach. Example ST dataset can be acquired from https://huggingface.co/datasets/MahmoodLab/hest.
